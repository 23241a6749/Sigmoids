### KiranaLink | Full-Stack Developer
**Tech Stack:** React, TypeScript, Node.js, Express, MongoDB, Firebase, Socket.io, Google Gemini AI

Engineered an end-to-end retail management SaaS platform designed to digitize operations for Kirana stores and MSMEs, serving as a comprehensive solution for inventory, billing, credit management, and customer engagement.

- **Authentication & Security:** Implemented JWT-based authentication with Google OAuth 2.0 integration using Passport.js
- **IndexedDB Integration:** Utilized Dexie.js for offline-first data persistence, enabling continued operation during network outages
- **Responsive UI/UX:** Crafted modern, mobile-responsive interfaces using Tailwind CSS, Framer Motion animations, and Lucide icons

**Technical Highlights:**
- Architected RESTful APIs with Express.js and MongoDB for scalable data management
- Implemented real-time bidirectional communication using Socket.io for live updates
- Integrated Firebase for cloud storage and real-time database synchronization
- Built custom React hooks for state management and reusable component logic
- Configured Vite build pipeline for optimized production deployments
- Utilized TypeScript for type-safe development across full stack

**Business Impact:**
- Streamlined retail operations by consolidating 10+ manual processes into one unified platform
- Enabled retailers to reduce inventory management overhead through automated tracking
- Facilitated cost savings through group buying feature connecting multiple retailers
- Improved customer relationships through digital credit tracking and WhatsApp-based ordering

## Interview Talking Points

### 1. **AI Integration Challenge**
"I integrated Google Gemini AI with WhatsApp Business API to process customer orders via voice and text. The challenge was handling multiple languages and voice notes - I implemented automatic language detection and used Gemini's multimodal capabilities to transcribe and understand orders, then emit real-time updates to the merchant dashboard via Socket.io."

### 2. **OCR Implementation**
"I built an OCR feature using Tesseract.js that allows shopkeepers to simply photograph supplier bills. The system automatically extracts product names, quantities, and prices. The tricky part was handling varying bill formats - I trained the system with preprocessing techniques and fallback validation to ensure accuracy."

### 3. **Group Buying Feature**
"I created a collaborative purchasing platform where multiple retailers can pool orders to get bulk discounts. I implemented real-time progress bars showing how many participants joined, with confetti animations when deals unlock. Upon completion, participants get a digital gate pass with QR codes for truck arrival verification."

### 4. **Real-time Architecture**
"I used Socket.io to enable real-time features across the platform - from tracking WhatsApp orders on the dashboard to showing live progress on group deals. This required careful event management and state synchronization between frontend and backend."

### 5. **Offline-First Design**
"Using Dexie.js (IndexedDB wrapper), I built offline capabilities so shopkeepers can continue billing even without internet. Data syncs automatically when connection restores, ensuring zero downtime in rural areas with unstable connectivity."
